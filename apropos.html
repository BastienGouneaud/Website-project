Ce site a été réalisé dans le cadre d'un projet pour l'UBFC

Voici la courbe représentative d'une fusée au lancement. On peut apercevoir que sa trajectoire est courbée puisque, pour atteindre l'orbite basse, il faut atteindre une vitesse horizontale d'environ 7600m/s. Pour tracer cette courbe nous avons utilisé une courbe de bézier avec pour point de départ les coordonnées (0,300), un point de contrôle avec pour coordonnées (0,200), puis un autre avec les coordonnées (200,150) et un point final avec pour coordonnées (750,75).
